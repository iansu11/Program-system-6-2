📚 Firebase 設定教學

步驟一：建立 Firebase 專案

1.請先前往 Firebase Console 並登入您的 Google 帳號，點擊「建立新的Firebase專案」。

2.輸入完專案名稱並按下「繼續」。

3.開啟 啟用 Gemini in Firebase 並按下「繼續」。

4.開啟 啟用這項專案的 Google Analytics 功能，並按下「繼續」。

5.選取 Default Account in Firebase 並按下「建立專案」。

6.等待完成後按下繼續專案即建立完成。

步驟二：在專案裡新增應用程式

1.回到專案後在專案名稱的下方，點擊「新增應用程式」的按鈕。

2.點擊網頁圖示，來新增一個應用程式。

3.接著會看到註冊應用程式的畫面，不要勾選Firebase託管，並且按下「註冊應用程式」的按鈕。

4.再來會看到新增 Firebase SDK ，請選擇使用 script 標記。

並且將 const firebaseConfig = { ... }; 裡面的內容複製起來，完成後按下前往控制台即完成新增。

5.回到控制台。

步驟三：Authentication設定。

1.先點選安全性，並選擇Authentication。

2.選擇開始啟用。

3.選擇登入方式，第一個先選電子郵件，第二個在選google。

4.電子郵件只需要啟用上面那個，底下的不用，啟動完成後按下儲存。

5.google 啟動後記得選取專案支援的電子郵件，完成後也是按儲存。

6.看到電子郵件與google皆顯示為已啟動即代表設定完成。

步驟四：firestore 設定

1.先選擇資料庫與空間，並選擇firestore。

2.選擇建立資料庫。

3.選取版本 Standard版，並且按下一步。

4.選擇位置 Taiwan，並且按下一步。

5.選擇以測試模式啟動，並且按下建立。

6.回到了資料庫的主控台，點選上方的規則。

7.將規則進行更改，改成以下這段規則，完成後按下發布所有設定便更改完成。

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if request.auth != null; } } }

步驟五：授權網路設定

1.點選Authentication的設定，裡面的授權網域並按下新增網域。

2.將解題系統的網址填入即設定完成。

步驟六：將金鑰填入到註冊區

用大括號將firebaseConfig裡面的內容複製並貼上即可(記得每個代碼冒號前面的標題要使用""括號起來，最外面再使用一個大括號)。

前面沒複製到的請依照以下步驟尋找回來。

1.點選左側欄位設定裡面的一般。

2.跳到一般設定往下滑就可以看到firebaseConfig的內容了。
